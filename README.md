## Study Workflow

이 repo는 공개 가능한 기술 자료만 다룬다.

학습 흐름은 `Discover -> Gate -> Scan -> Analyze -> Document -> Explain -> Connect -> Close`다.

| 단계 | 산출물 |
|---|---|
| Gate | public/private 판단 |
| Scan | 읽을 가치와 우선순위 |
| Analyze | 깊은 분석 |
| Document | GitHub Issue 본문 |
| Explain | 쉬운 설명 댓글 |
| Connect | 관련 이슈 연결 |

세부 규칙은 `CLAUDE.md`와 `.claude/skills/`를 따른다.
